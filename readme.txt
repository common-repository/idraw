== Description ==

idraw allows you to create beautiful diagrams and insert them into your posts or pages directly. It supports many kinds of diagrams: flowcharts, UML, circuit diagrams, formula, etc. It has the following features.

1.It is powerful. You can draw flowchart, UML, circuit diagrams, formula, chart and other diagrams with idraw.

2.It is beautiful. Gradient color fill and shadow are supported.

3.It is fast. It is developed with js + html5 canvas, it is fast to load and run.

4.It is easy to use. Many shortcuts and context menus make it easy to use.

5.It is free and without registration.

== Installation ==

1. Upload idraw.x.y.zip to the `/wp-content/plugins/` directory and unzip it here.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure wp-content/uploads is writable.

== Usage ==

1. Click the idraw button in post/page editor to open diagram editor.
2. Create diagrams with some drag & drop.
3. Click "Exit" button, confirm it with "yes" to save and upload image to blog server.
4. Close image dialog return back to post/page editor.
